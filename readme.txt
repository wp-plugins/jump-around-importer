Plugin to easy import Flick pictures with a certain tags to wordpress and create a draft

== Description ==

The Plugin is an extension to the "WordPress Flickr Manager" by [tgardner](http://tgardner.net/wordpress-flickr-manager/ to easily import
picture from flickr by given tag. It groups the picture by geoposition and creates a draft for each group, title, description, tags and 
date prefilled. The images are created as meta field imageX as well the geo data, so it works smothly with the 
[wordpress geo mashup](http://code.google.com/p/wordpress-geo-mashup)  Plugin.
